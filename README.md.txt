# Генератор QR-кодов

## Описание проекта
Консольное приложение на Python для генерации QR-кодов из текста или URL-адресов.

## Функциональность
- Генерация QR-кода из любого текста или URL
- Настройка параметров: версия, уровень коррекции ошибок, размер модуля
- Автоматическое создание имени файла с датой и временем
- Обработка конфликта имён (добавление индекса)
- Валидация ввода (проверка на пустоту и длину)
- Сохранение истории операций в JSON
- Просмотр истории через консольное меню

## Технологии
- Python 3.8+
- qrcode
- Pillow (PIL)
- JSON
